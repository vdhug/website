Engenharia Vivace website, an Electrical Engineering company.

Engenharia Vivace

Engenharia Vivace is an Electrical Engineering company, founded in 2015, offers maintenance of electrical, hydraulic and refrigeration systems, elaboration of projects in the different areas of civil and eletrical engineering and products and services for the generation of energy from renewable sources.

Hosting

Amazon Web Services was used to host the website because is reliable, fast, secure and has a really fair price. Beyond that AWS offers a really easy way to scale your website.

Animations

JavaScript was used to manage the animations in the website. The animations was part of a really famous library animate.css, created by Daniel Eden

Templates

PHP was used to transform static html pages into dynamic web pages and to allow the administrators of the website add, edit or remove informations through the CMS Wordpress.

Style

CSS was used to style all the website. All the style in the website was produced specific to this project.
